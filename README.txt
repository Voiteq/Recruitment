Voiteq Coding Exercise
######################

This is a very poor implementation of a command line application that can find if two supplied words are anagrams of one another.

Your challenge is to refactor this code with a view to making it maintainable, understandable and testable. There are also some improvements that could be made to the algorithm.

There are some skeleton NUnit tests. Make these pass and feel free to expand with your own.

You have 30 minutes to complete this task.
